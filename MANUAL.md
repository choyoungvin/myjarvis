## 1. 프로그램 개요

**J.A.R.V.I.S (Just A Rather Very Intelligent System)**는 마블의 아이언맨에 등장하는 AI 비서 '자비스'에서 영감을 받아 제작된 **대학생 전용 학업 비서 프로그램**입니다.

단순한 유틸리티 도구가 아닌, **"내가 매일 아침 필요한 정보를 한 번에 브리핑해주는 AI 비서"** 를 목표로 설계하였습니다.

---

## 2. 주요 기능

### 2.1 종합 브리핑 모드 — "안녕 자비스"
음성으로 "안녕 자비스"라고 말하면 자비스가 다음 정보를 순서대로 보고합니다:

1. 📅 **오늘의 날짜 및 요일**
2. ⏰ **현재 시각**
3. 🌤 **실시간 날씨 정보** (서울 기준, wttr.in API 활용)
4. 📚 **오늘의 수업 일정**
5. ⚠ **마감 임박 과제 알림**

### 2.2 오늘의 수업 일정 (좌측 패널)
- 오늘 요일에 해당하는 수업 목록을 자동으로 표시
- 현재 수업 중인 강의는 초록색으로 강조 표시
- 이미 끝난 수업은 흐리게 표시

### 2.3 과제 마감 D-Day 관리 (우측 패널)
- 등록된 과제의 마감일까지 남은 날짜(D-Day)를 실시간 계산
- **D-3 이하**: 빨간색 긴급 알림
- **D-7 이하**: 주황색 주의 알림
- **D-8 이상**: 파란색 일반 표시

### 2.4 할 일 목록 (To-Do List)
- 음성 명령 또는 텍스트 입력으로 할 일 추가
- 체크박스로 완료 처리 / 삭제 기능
- 브라우저 `localStorage`에 저장 → 창을 닫아도 유지

### 2.5 음성 반응 홀로그램 시각화
- Web Audio API를 활용하여 사용자 목소리의 주파수 데이터를 실시간 분석
- 목소리 크기에 따라 아크 리액터 형태의 홀로그램이 **방사형 파형(Radial Waveform)** 으로 반응
- 아이언맨 HUD 스타일의 회전 링, 네온 글로우 효과 적용

## 4. 실행 방법

본 프로그램은 두 가지 방식으로 실행할 수 있습니다:

### 방법 A: Google Chrome 브라우저 실행 (권장 ⭐)
**음성 인식 기능이 가장 안정적으로 작동하는 방식입니다.**
1. `jarvis-ai/app/index.html` 파일을 마우스 오른쪽 버튼으로 클릭합니다.
2. **[다음으로 열기] > [Google Chrome]**을 선택합니다.
3. 중앙의 **[ACTIVATE VOICE LINK]** 버튼을 클릭하고 마이크 권한을 허용합니다.

### 방법 B: Electron 데스크탑 앱 실행
**macOS 캘린더 및 노션 연동 기능을 사용할 때 필요합니다.**
1. 터미널(Terminal) 앱을 엽니다.
2. 프로젝트 폴더로 이동 후 `npm start` 명령어를 입력합니다.
   > ⚠ **주의**: Electron 환경에서는 Google의 보안 정책상 음성 인식(`network` error)이 불안정할 수 있습니다. 이 경우 방법 A를 사용하세요.

## 5. AI Agent 협업 과정

본 프로그램은 **Antigravity AI Agent**와의 대화를 통해 기획, 설계, 구현되었습니다.

### 5.1 기획 단계
- AI와의 대화를 통해 "컴퓨터 제어 도구"에서 **"학업 비서"** 컨셉으로 방향 전환 결정
- 교수님의 과제 가이드라인(확장자 변환, 프로세스 관리 등)을 "음성 명령으로 실행하는 자비스" 컨셉으로 승화

### 5.2 기술 설계
AI Agent와 함께 결정한 핵심 기술 선택:
- **Web Audio API** → 마이크 입력을 주파수 데이터로 변환, Canvas로 홀로그램 파형 생성
- **Web Speech API** → 브라우저 내장 한국어 음성 인식 엔진 활용
- **wttr.in API** → API 키 없이 사용 가능한 무료 날씨 정보 제공
- **localStorage** → 서버 없이 할 일 목록을 로컬에 영구 저장

### 5.3 구현 및 개선
- 1차: 기본 홀로그램 UI 및 오디오 시각화 구현
- 2차: 음성 인식 엔진 연결 및 명령어 처리 로직 추가
- 3차: 학업 비서 기능(일정, 과제, 날씨, 브리핑) 전면 도입
- 4차: 디자인 완성도 강화 (아크 리액터 링, HUD 레이아웃, D-Day 배지)

## 7. 문제 해결 (FAQ)

**Q: "Recognition error: network" 오류가 발생합니다.**
A: 이 오류는 주로 **Electron 앱**에서 실행할 때 발생합니다. Google 음성 인식 서비스는 보안상 공식 브라우저(Chrome)가 아닌 환경에서의 접근을 제한할 수 있습니다. 
*   **해결책**: `app/index.html` 파일을 **Chrome 브라우저**에서 직접 열어 실행해 주세요. 브라우저 모드에서도 날씨, 일정, 과제 계산 기능은 모두 정상 작동합니다.

**Q: 마이크 파형이 움직이지 않아요.**
A: 마이크 권한이 거부되었거나, 시스템 설정에서 마이크 입력 장치가 올바르게 선택되지 않았을 수 있습니다. 브라우저 주소창 왼쪽의 자물쇠 아이콘을 눌러 권한을 확인해 주세요.
