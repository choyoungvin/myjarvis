## 🌟 주요 특징 (Key Features)

- **HUD-Inspired Hologram UI**: 아이언맨의 아크 리액터와 HUD 테마를 적용한 프리미엄 다크 모드 디자인.
- **Voice-Responsive Waveform**: `Web Audio API`를 사용하여 사용자의 목소리 크기와 주파수에 따라 실시간으로 반응하는 캔버스 애니메이션.
- **AI Voice Command Engine**: `Web Speech API`를 통해 한국어 음성 명령을 인식하고 시스템 기능을 수행.
- **Academic Utility Integration**:
    - **Process Hub**: 실행 중인 앱 목록 확인 제안.
    - **File Standardizer**: 확장자 일괄 변경 로직 안내.
    - **Focus Timer**: 학업 집중을 위한 포모도로 타이머 기능.
    - **Academic Logic**: 계산기, 날짜, 시스템 정보 확인 등.

## 🛠 사용된 기술 (Tech Stack)

- **Frontend**: HTML5, CSS3 (Glassmorphism, HUD Design), JavaScript (ES6+)
- **Audio Engine**: Web Audio API (AnalyserNode, Canvas Drawing)
- **Speech Engine**: Web Speech API (webkitSpeechRecognition)

## 🚀 시작하기 (How to Use)

1. 이 저장소를 클론(Clone)합니다.
2. `app/index.html` 파일을 구글 크롬(Google Chrome) 브라우저로 엽니다.
3. 화면 중앙 하단의 **[ACTIVATE VOICE LINK]** 버튼을 클릭합니다.
4. 마이크 권한을 허용합니다.
5. "자비스, 안녕" 또는 "시간 알려줘"와 같은 명령어로 대화를 시작하세요!
